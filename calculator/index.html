⚠️

計算上の注意事項

  • Abundant〜Mythic は Fandom Index の公開 rarity table に基づく既定値です。
  • Exotic は同じ table の公開値が未記載のため、Trash 9% と他 tier の公開値を用いて合計 100% になるよう 3.1% として推定しています。
  • 釣り場は自動選択です。候補ごとに期待値/時間が最も高い場所を採用しています。
  • 時間帯は固定せず、朝 / 昼 / 夕方 / 夜を同じ比率で平均しています。
  • 天気は固定せず、晴れ / 雨 / Moonrain / 嵐 / 霧を同じ比率で平均しています。
  • 同じレア度の魚は、公開されている索引に載っている魚をいったん同じ出やすさとして配分しています。
  • Luck / Big Catch Rate / Max Weight の効き方は公開情報だけでは足りないため、期待値にはまだ推定が入っています。
  • 逃がしやすさはレア度ごとの近似です。魚ごとの正確な内部式は公開されていません。
  • 価格レンジ未取得の魚が 1 種あります。期待値はそれらを 0G 扱いした下限値です: Venomous Watcher

各魚の条件や売値レンジ、釣獲率、期待値を確認できます。

  • • いま見ている条件: 釣り場 Tanglewood Bog、時間帯 平均で見る、天気 平均で見る
  • • 計算に入っている魚: 10 種 (価格情報が未取得の魚 1 種を除く)
  • • どの魚が釣れるか: 各釣り場の魚リストから、時間帯と天気の条件に合う魚を選んでいます。
  • • 装備の効果: 選んだ Rod / Line / Bobber / Enchant のステータスを合算しています。
  • • レア度の影響: 公開されているレア度テーブルを基準に、同じレア度の魚は同じ確率で出るように計算しています。
  • • Luck の効果: 0.50x の補正として反映しています。(※内部計算式は未公開のため推定値)
  • • Big Catch / Max Weight の効果: 魚の重さを上位 5% の範囲に寄せる形で反映しています。(※推定式)
  • • 見た目・サイズボーナス: 約 1.242x のボーナスとして計算に入れています。(※コミュニティ観測値に基づく推定)
  • • 期待値/回: 各魚の釣れる確率 × 推定売値 × 見た目・サイズボーナスを全魚種で合計した値です。
  • • 期待値/時間: 期待値/回 × (1時間 ÷ 1回の所要時間) で計算しています。
  • • 使用中のレア度重み: Abundant 5.40, Common 9.00, Curious 5.58, Elusive 7.15, Fabled 2.31, Mythic 0.94, Exotic 0.62
